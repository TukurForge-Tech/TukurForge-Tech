Términos de Servicio y Reglas de Uso

Reglamento Operativo de la Plataforma | SimuTukur

Las presentes normas rigen el uso técnico y operativo de la plataforma SimuTukur. Su aceptación es obligatoria para la utilización del servicio.

1. Requisitos Técnicos y Responsabilidad del Usuario

Para un rendimiento óptimo, EL SISTEMA requiere ser ejecutado en un equipo de cómputo (PC o Laptop) con un navegador web moderno actualizado (Google Chrome, Edge, Safari) y una conexión a internet de banda ancha estable. SimuTukur no se hace responsable por fallas de conectividad local del usuario, desconexiones, cortes de energía eléctrica o incompatibilidad de hardware local que impidan la correcta culminación de un simulacro.

2. Política Antifraude y Uso de Cuenta

Las credenciales de acceso son estrictamente personales e intransferibles. El sistema cuenta con herramientas de monitoreo de direcciones IP y sesiones. Si se detectan intentos de acceso simultáneo desde diferentes ubicaciones geográficas, o patrones de compartición de contraseñas, la cuenta será suspendida inmediatamente de forma permanente sin derecho a reembolso.

3. Uso Aceptable de la Inteligencia Artificial

El "Tutor IA" es una herramienta de asistencia educativa diseñada para brindar explicaciones y pistas. Queda prohibido el uso de ingeniería de comandos (prompt injection) para intentar vulnerar el sistema, extraer respuestas directas del código fuente, o usar el motor de IA para resolver exámenes ajenos a la plataforma. El abuso del sistema de Energía IA resultará en la restricción del chat.

4. Derechos de Propiedad y Prevención de Piratería

El usuario se compromete a no utilizar bots, scripts automatizados, extensiones de raspado de pantalla (web scraping) ni software de grabación de pantalla con el fin de extraer la base de datos de reactivos. Cualquier intento de violar la seguridad del sitio web o sobrecargar los servidores (ataques DDoS) será reportado a las autoridades cibernéticas correspondientes.

5. Modificaciones al Servicio

TukurForge Tech se reserva el derecho de actualizar, modificar o descontinuar temporalmente partes de la interfaz, el contenido de los exámenes o los algoritmos de IA para aplicar mejoras de rendimiento, sin previo aviso a los usuarios, garantizando siempre mantener el núcleo del servicio contratado.
